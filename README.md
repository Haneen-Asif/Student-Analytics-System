# 🎓 Student Analytics System

A Flask-based web application that allows users to manage student records, calculate grades, and view analytics in a clean dashboard interface.

---

## Features

- ➕ Add student records
- 🗑 Delete students
- 📊 Grade calculation
- 📈 CGPA calculation
- 📋 Dashboard view
- 🎨 Professional UI with Bootstrap

---

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Bootstrap
- SQLite (or your database)
